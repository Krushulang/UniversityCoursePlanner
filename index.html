This application can be used by University of Washington Bothell students to find recommendations on what courses could be a good fit in their schedule. It does this through a vote based algorithm. Note that this application is for suggested courses and not the only opinion a student should get. Please consult an academic advisor for any further questions about courses that may be considered.

To begin, select a campus, a planned or admitted major, and classes taken at UW. Additional constraints can be added to the query for better results. Some are required while some are optional.

The results will be in the form of a ranked list that will show what course is best to take next quarter (#1) given the constraints, and the classes that may also be considered (#2-10). The application will try to find what is best through the voting algorithm. For a full set of classes to take, try multiple different queries by changing aspects like time, days, etc.

Optional filters include:
  • Time
  • Days
  • In-person/Online
  • Size
  • Education requirements
  • Curriculums
